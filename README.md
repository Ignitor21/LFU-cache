## Installing project:
1. Clone this repository to your computer:
```
    git clone git@github.com:Ignitor21/LFU-cache.git
    cd LFU-cache
```
2. Run cmake
```
    cmake -B build
    cd build/[cache_name] 
    cmake --build .
```

[cache_name] is `lfu` or `perfect`

3. ./[cache_name]

## Tests
